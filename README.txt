Nome do Jogo: Chicken Fight

Identificação
Diogo Martins/ nº 21091
José Alves/ nº 21081

Controlos

W- Mover para cima
S- Mover para baixo
A- Mover para a esquerda
D- Mover para a direita
F/Rato esquerdo- Disparar a arma
G/Rato direito- Ataque corpo-a-corpo

Descrição
Chicken Shooter é um jogo onde o objetivo é acumular o maior numero de pontos possivel. Isto é feito através do massacre de galinhas, que são neste caso, os nossos inimigos. Cada abate resulta em 1 ponto. O jogador começa com uma de três armas entre as quais este roda aleatoriamente a cada 10 segundos. Ao trocar de arma, o jogador regenera tambem 1 ponto de vida, tendo um máximo de 7. A vida das galinhas aumenta com o tempo de jogo.

Alguns detalhes de implementação

-Tanto as galinhas como o jogador têm um pequeno framebuffer de 0.5 segundos após serem atingidos para evitar dano excessivo
